## Tech Stack
- Language: C# / .NET Framework 4.7.2
- Database: MS-SQL Server (SQL Express)
- ORM: Entity Framework 5.0
- IDE: Visual Studio 2022

## How to Run
1. Restore NuGet packages
2. Update connection string in `App.config` to point to your SQL Server instance
3. Run `database/schema.sql` on your SQL Server
4. Press F5 in Visual Studio

## Project Overview
UMS2026 is a desktop-based University Management System built with Windows Forms. It supports three user roles — Administrator, Instructor, and Student — each with a dedicated dashboard after login. The system manages departments, courses, instructors, students, enrollment records, class scheduling, and basic reporting.
